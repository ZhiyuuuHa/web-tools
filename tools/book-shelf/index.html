📝 如何添加书籍？

1. 将书籍文件放入 tools/book-shelf/books/ 目录

2. 在 script.js 的 booksData 数组中添加书籍信息：

{
    title: "书籍名称",
    author: "作者",
    format: "pdf",  // pdf / epub / mobi / other
    cover: "covers/book-cover.jpg",  // 封面图片路径（可选）
    file: "books/filename.pdf",  // 文件路径
    description: "简短描述"
}
